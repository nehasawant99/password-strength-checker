# Password Strength Checker

## Overview
This Python-based cybersecurity project checks password strength using regular expressions and common password security standards.

The tool analyzes passwords and provides suggestions to improve security.

---

# Features
- Minimum length validation
- Uppercase letter detection
- Lowercase letter detection
- Number detection
- Special character detection
- Password strength rating
- Security improvement suggestions

---

 Technologies Used
- Python
- Regular Expressions (re module)

---

# How It Works

The program checks:
- Password length
- Uppercase characters
- Lowercase characters
- Numeric digits
- Special symbols

Based on these checks, the password is classified as:
- Weak
- Medium
- Strong
- Very Strong

- Unauthorized access
- Credential theft
- Account compromise
- Data breaches
