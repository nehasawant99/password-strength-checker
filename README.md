# Password Strength Checker

## Overview
This Python-based cybersecurity project checks password strength using regular expressions and common password security standards.

The tool analyzes passwords and provides suggestions to improve security.

---

# Features
- Minimum length validation
- Uppercase letter detection
- Lowercase letter detection
- Number detection
- Special character detection
- Password strength rating
- Security improvement suggestions

---

 Technologies Used
- Python
- Regular Expressions (re module)

---

# How It Works

The program checks:
- Password length
- Uppercase characters
- Lowercase characters
- Numeric digits
- Special symbols

Based on these checks, the password is classified as:
- Weak
- Medium
- Strong
- Very Strong

---

# Example Output

Enter your password:
Admin123

Password Strength: Medium

Suggestions:
- Add at least one special character.

---

# Cybersecurity Importance
Weak passwords are one of the leading causes of:
- Unauthorized access
- Credential theft
- Account compromise
- Data breaches

Strong passwords help improve account and system security.
